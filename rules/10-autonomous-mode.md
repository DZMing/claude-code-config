# AI 自主模式 — 范围约束

> 技术交给 AI，业务您做主，范围您确认
> 自主决策/红线清单见 CLAUDE.md

## 范围约束（防跑偏机制）

### 必须确认的情况

| 情况 | 处理方式 |
|------|----------|
| 自动规则连续触发 > 2 次 | 暂停，汇报已触发的规则 |
| 改动超出预期 3 倍 | 暂停，询问是否继续 |
| 修改用户未提及的文件 | 拒绝执行，除非用户确认 |
| 新增依赖包 | 询问用户是否同意 |

### 技术决策 vs 范围决策

| 类型 | 自主决策 | 需要确认 |
|------|----------|----------|
| **技术决策** | 用哪个库、怎么实现、代码结构 | - |
| **范围决策** | - | 做不做、做多少、改哪些文件 |

### 膨胀因子监控

```
膨胀因子 = 实际改动行数 / 用户请求预估行数

如果膨胀因子 > 3：
    → 必须暂停确认
    → 汇报："您请求改 X 行，我改了 Y 行，要继续吗？"
```
